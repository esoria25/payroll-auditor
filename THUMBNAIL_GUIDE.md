## Specifications:
- **Size**: 1280x720 pixels (16:9)
- **Format**: JPG or PNG
- **File size**: Under 2MB

## Design Elements:

### Colors:
- Background: Dark blue or gradient (#1a1a2e to #16213e)
- Text: White (#ffffff)
- Accent: Green (#00ff88) or Blue (#00d4ff)

## Template Ideas:

### Option 1: Results Focus
- Large screenshot of differences table
- "8 Differences Found" in big text
- Green checkmark icon

### Option 2: File Types
- Three file icons (CSV, Excel, PDF)
- Arrows pointing to comparison result
- "Compare Any Format"

### Option 3: Speed Focus
- Stopwatch icon
- "Results in Seconds"
- Clean, minimal design

## Text Guidelines:
- Use max 6-8 words
- High contrast (white on dark)
- Large, readable font
- No small text (won't be readable)
