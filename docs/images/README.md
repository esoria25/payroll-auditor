# Screenshots

To add screenshots to the main README:

1. Take screenshots of:
   - Web interface (upload page)
   - Comparison results page
   - Summary statistics view

2. Save them as:
   - web-interface.png
   - upload.png
   - results.png
   - summary.png

3. Place them in this directory

4. They will automatically appear in the main README

## Quick Screenshot Guide

### For macOS:
- Cmd + Shift + 4: Select area to capture
- Cmd + Shift + 4, then Space: Capture window

### Recommended Screenshots:
1. **web-interface.png**: Full view of http://localhost:8080
2. **upload.png**: File upload area with drag-and-drop
3. **results.png**: Comparison results with differences highlighted
4. **summary.png**: Summary statistics section
