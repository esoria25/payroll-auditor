## Before You Start:
- [ ] Screen recording software ready (OBS/QuickTime)
- [ ] Audio tested (microphone working)
- [ ] Desktop cleaned up
- [ ] Browser in clean profile
- [ ] Terminal ready with clean history
- [ ] Sample data generated
- [ ] Server can start successfully
- [ ] Script visible on second monitor

## Recording Settings:
- Resolution: 1920x1080
- Frame rate: 30 fps
- Audio: 44.1kHz or 48kHz

## Scenes to Record:

### Scene 1: GitHub Page (15s)
- [ ] Navigate to repo
- [ ] Show features
- [ ] Highlight stars/badges

### Scene 2: Installation (30s)
- [ ] Terminal: git clone
- [ ] Terminal: pip install
- [ ] Terminal: generate sample data

### Scene 3: Web Interface (60s)
- [ ] Start server
- [ ] Open browser
- [ ] Upload first file
- [ ] Upload second file
- [ ] Click compare
- [ ] Show results (scroll through)

### Scene 4: Command Line (25s)
- [ ] Run CLI command
- [ ] Show terminal output
- [ ] Open HTML report

### Scene 5: Closing (10s)
- [ ] Back to GitHub
- [ ] Show documentation
- [ ] End screen with CTA
